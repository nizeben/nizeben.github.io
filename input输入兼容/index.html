input输入法测试

发表于 2017-07-27

浏览器 input输入

对以下问题进行不同浏览器的测试：

输入法为中文输入时：

1.输入过程中是否会触发input事件

2.输入过程中，点击回车输入英文时是否会触发input

3.点击空格输入完毕时会否触发input

4.ie下是否能触发propertychange

测试浏览器：
chrome Firefox qq 搜狗 ie8 ie11
测试输入法：
搜狗拼音输入法
mac原生输入法
微软拼音输入法
测试结果：

1.ie8中不支持input事件

2.输入过程中，点击回车输入英文时是否会触发input

3.ie8下会触发propertychange 但是微软拼音输入法会触发两次

Mac

1.chrome 正常
2.Firefox 正常

Windows7

微软输入法
1.chrome正常
2.Firefox正常
3.ie8正常
4.ie9 用delete删除时无法触发input，但可以用keyup来代替（keydown比keyup每次输入少一个字符）

搜狗输入法
1.ie8正常（输入时不会进入输入框）
